# TRUEQUE-ACTIVO

TRUQUE-ACTIVO es un servicio web HTML + PHP para el intercambio entre productos.

## Instalación

Descargue los archivos [pip](https://pip.pypa.io/en/stable/) para instalar TRUEQUE-ACTIVO en su localhost.

```bash
pip install foobar
```
